# Roadmap

## v0.2.0

- Offline deterministic demo for candidate scoring and rejection.
- CI validation badge.
- JSON report output for curation results.
- Safer install dry-run mode.
- More unsafe-skill fixtures.

## v0.3.0

- Curated skill index format.
- Skill conflict detector.
- Cross-platform install path tests.
- More risk model examples.

## Not Planned

- Auto-installing high-star skills without review.
- Endorsing third-party skills as safe.
- Replacing human approval for risky installs.
