# Social Preview Copy

GitHub social preview is configured in repository settings, not by README alone.
Use the existing `assets/hero.png` as visual direction, then upload a dedicated
1280 x 640 PNG/JPG preview in GitHub Settings.

## Recommended text

```text
GitHub Skill Curator
Find • Score • Safety-scan • Install Agent Skills
Review before install
For Codex / Claude Code
```

## Visual direction

- large magnifying glass over `SKILL.md` folders;
- three candidate cards: `Recommend`, `Warn`, `Reject`;
- small safety score or shield element;
- keep text sparse so it survives link previews.

## Avoid

- claims that third-party skills are guaranteed safe;
- tiny README screenshots;
- dense command output;
- generic AI art that does not show skill discovery or review.
